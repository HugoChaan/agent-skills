---
title: 对话式智能体 API 概述
---

本文介绍声网对话式智能体服务 RESTful API 的基本信息。

## 数据格式

所有 HTTP 请求头部的 Content-Type 类型为 `application/json`。所有请求和响应内容的格式均为 JSON。

## 接口列表

| 方法 | 路径 | 描述 |
| --- | --- | --- |
| POST | `/v2/projects/{appid}/join` | [创建对话式智能体](./start-agent) |
| POST | `/v2/projects/{appid}/agents/{agentId}/leave` | [停止对话式智能体](./stop-agent) |
| POST | `/v2/projects/{appid}/agents/{agentId}/update` | [更新智能体配置](./agent-update) |
| GET | `/v2/projects/{appid}/agents/{agentId}` | [查询智能体状态](./query-agent-status) |
| GET | `/v2/projects/{appid}/agents` | [获取智能体列表](./get-agent-list) |
| POST | `/v2/projects/{appid}/agents/{agentId}/speak` | [播报自定义消息](./agent-speak) |
| POST | `/v2/projects/{appid}/agents/{agentId}/interrupt` | [打断智能体](./agent-interrupt) |
| GET | `/v2/projects/{appid}/agents/{agentId}/history` | [获取智能体短期记忆](./get-history) |
